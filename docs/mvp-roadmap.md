# MVP Roadmap

## Phase 1 - Foundation

- Monorepo setup.
- Prisma schema.
- Express app shell.
- Auth service with JWT and refresh tokens.
- Shared constants and validators.
- Dashboard summary contract.

## Phase 2 - Work Hours

- Work shift CRUD.
- Worked-hour calculation.
- Income calculation.
- Weekly and monthly summaries.
- Configurable Germany work-limit policy and warning output.

## Phase 3 - Expenses

- Expense CRUD.
- Monthly budget CRUD.
- Monthly expense summary.
- Category summary.
- Savings calculation using work income and expenses.

## Phase 4 - Classes, Tasks, Reminders

- Class schedule CRUD.
- Weekly calendar.
- Task CRUD and complete endpoint.
- Central reminders.
- Today's dashboard aggregation.

## Phase 5 - Groceries

- Grocery item CRUD.
- Shopping list CRUD.
- Purchase recording.
- Price history and trend.
- Expected finish date reminders.

## Phase 6 - Cleaning Habits

- Cleaning task CRUD.
- Complete action.
- Days-since-last-done calculation.
- Interval reminders.

## Phase 7 - AI Assistant

- AI provider interface.
- Ollama provider.
- OpenAI provider placeholder.
- Expense advice.
- Weekly summary.
- Grocery savings.
- Study plan.
- Work-limit warning suggestions.

## Phase 8 - Hardening

- Unit tests for calculation services.
- Integration tests for major routes.
- Rate limiting and security headers.
- Production environment docs.
- Web and mobile polish.
